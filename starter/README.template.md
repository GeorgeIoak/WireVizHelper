# __PROJECT_NAME__

Cable drawing project generated by WireVizHelper.

## Quick Build

**Option 1: Windows App (EXE)**

```powershell
path\to\WireVizHelper.exe build __YAML_FILE__
```

Output files are generated in `output/`.

## Prerequisites

- `Graphviz` must be installed for WireViz diagram generation.
- PDF output uses a Chromium-based browser print engine by default.

## Typical Workflow

1. Edit `__YAML_FILE__` as the source of truth.
2. Add supporting images to `images/`.
3. Rebuild with `build.py`.
4. Review outputs in `output/`.

## Output Summary

- `output/__YAML_STEM__.html`
- `output/__YAML_STEM__.svg`
- `output/__YAML_STEM__.png`
- `output/__YAML_STEM__.pdf` (when browser PDF export succeeds)
- `output/__YAML_STEM__.bom.tsv`

## YAML and BOM Notes

- Use WireViz-native BOM fields (`manufacturer`, `mpn`, `pn`, `supplier`, `spn`).
- For product photo-in-row behavior, use `additional_bom_items` and set image HTML in `spn`.
- WireVizHelper renames BOM header `SPN` to `Product Photo`.

For full YAML options and metadata layout controls, see the helper repository docs:

- `/path/to/WireVizHelper/docs/yaml-options.md`
